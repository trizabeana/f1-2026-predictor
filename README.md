# 🏎️ F1 2026 Predictor: Data-Driven Performance Forecasting

A data-driven predictive model that analyzes F1 2026 pre-season testing data to forecast team and driver competitiveness for the opening Grand Prix.

## 🔬 Methodology & Scoring
Built with **Python**, the model combines three key performance indicators into a weighted scoring system to generate a competitiveness ranking per team:
* **Reliability:** Total laps completed during testing sessions.
* **Raw Pace:** Analysis of the absolute fastest lap times.
* **Consistency:** Calculation of lap time variance across long stints.

## 💻 Interface & Visualization
The project features an interactive dashboard built with **Streamlit**, allowing real-time exploration of pre-season insights. Results are visualized with **Plotly**.

## 🛠️ Tech Stack
* **Framework:** Streamlit
* **Data Analysis:** Pandas, NumPy
* **Visualization:** Plotly
* **Data Source:** FastF1 API
